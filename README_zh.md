HIGHFLIP
===================
HIGHFLIP是一个顶层联邦学习互通服务，用于解决异构联邦平台间相互通信的问题。名字来自于英文**HIGH** 
layer **F**ederated **L**earning **I**ntercommunication **P**rotocol的缩写. 

# 项目介绍
HIGHFLIP项目是用于解决异构的联邦学习平台间隔
